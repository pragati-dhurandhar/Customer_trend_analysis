## üìå Project Workflow
1. Data Ingestion & Cleaning (Python)

Established a connection between Python and a SQL Database (MySQL/PostgreSQL) using SQLAlchemy.

Developed complex SQL scripts to answer critical business questions, such as:
Customer Loyalty: Calculating the Lifetime Value (LTV) of subscribed vs. non-subscribed users.

Regional Performance: Ranking regions by total sales volume and average order value (AOV).

4. Visualization & Business Intelligence (Power BI / Seaborn)
